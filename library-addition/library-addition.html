LIBRARY ADDITION

A "Parasite" for Hunt Library at CMU

Project done in Prof Kai Gutschow's first year studio.

Being tasked to design an addition to the campus library at Carnegie Mellon, this project aims to push the 40 year old library towards a library of the future. The project uses a roof element which behaves like a parasite in the library, making its way around the library floors and changing the program to meet requirements. The design originated to create the preliminary mood of studying under a tree, and expanded to become an element which opens up and defines the library.
