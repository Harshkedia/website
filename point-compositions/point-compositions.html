Point Compositions

Mathematical Transformations of Points

Done under Prof Golan Levin.

Taking inspiration from D'arcy Thompson's Theory of Transformations, this project is an attempt to create beauty from basic geometric shapes, in this case, making Mattise's Dancers out of a simple square.

This project was heavily focused on computational craft with a requirement to use easing functions.
